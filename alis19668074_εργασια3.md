# Ηλεκτρονική εκδοτική, μάθημα Ε εξαμήνου
## Άσκηση 3 της θεωρίας
### Ειρήνη Μπρέντα, 19668074
 Σε αυτή την εργασία θα μελετήσουμε το συντακτικό του **Markdown** και θα δημιουργήσουμε ένα αρχείο κατασήμανσης.
Το Markdown είναι μια *απλή γλώσσα σήμανσης* με σύνταξη μορφοποίησης απλού κειμένου. Σε αυτήν μπορούμε να κάνουμε γραφή :
 - Με έντονα γράμματα
 - Με πλάγια γράμματα
 - Με υπογραμμισμένα γράμματα
 
 Επίσης είναι <ins>εφικτή η χρήση λίστας</ins>, αριθμημένης ή μη, ώστε να μπορεί να γίνει κατηγοριοποίηση των δεδομένων και του περιεχομένου μας.
 Επιπλέον διαβάζοντας τις σημειώσεις παρατηρείτε οτι μπορούμε να προσθέσουμε :
Link και εικόνες μέσα στο κείμενο :
 - είτε σαν αρχείο εικόνας
 - είτε με το link της εικόνας

Για την δημιουργία του περιεχομένου χρησιμοποίησα το ειδικό λογισμικό stackedit [stackedit](https://stackedit.io/app#)
